# Игра "Лабиринт Сокровищ"

Текстовая игра-квест, в которой игрок исследует лабиринт, решает загадки и стремится найти выход из лабиринта

Проект написан на Python и использует Poetry для управления зависимостями и виртауальным окружением.

Для установки игры, убедитесь, что у вас установлен Python и Poetry, а затем склонируйте репозиторий с помощью команды
git clone https://github.com/Seeneld/project1_priezzhikh_timofey_M25-555.git

Затем перейдите в папку проекта:
cd project1_priezzhikh_timofey_M25-555

Установите зависимости с помощью poetry и запустите проект:
poetry install
poetry run project

Справка доступна через ввод в консоль игры команды help
